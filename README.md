# Grails Quick Start

a simple Gradle quick start application for Gradle builds

See [John Engelman](https://github.com/johnrengelman)'s [Gradle Summit](http://gradlesummit.com/) talk, ["Evolving Grails Builds With Gradle"](https://youtu.be/3sWgExNwkR4) for a basic overview of automation with the [Grails Gradle plugin](https://github.com/grails/grails-gradle-plugin).

## requires

* java 7.+

## basic usage

* `git clone`
* `./gradlew test` — test Grails application
* `./gradlew run` — run Grails application
* `./gradlew eclipse` — bootstrap basic IDE integrations
* `./gradlew grails-<command>` — wrap-execute grails command
* `./gradlew tasks` — additional project tasks
